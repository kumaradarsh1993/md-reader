# docs/img — screenshots and GIFs for the README

This folder holds the visual assets the README references. See [DEMO.md](../../DEMO.md) at the repo root for the recording storyboard, software recommendations, and file-naming conventions.

Files the README expects (uncomment the corresponding image lines once each exists):

- `hero.gif` — 4–6s loop showing smart-edit in action. Headline asset.
- `live-reload.png` — Viewer with `📡 live` badge visible.
- `smart-edit.png` — Smart edit surface with floating selection toolbar.
- `tabs.png` — Tab bar with multiple files open.
- `side-panel.png` — Side panel with file browser + outline.

Keep total folder size under ~10 MB so `git clone` stays cheap. Optimise GIFs with [ScreenToGif](https://www.screentogif.com/)'s built-in compressor; optimise PNGs with [tinypng.com](https://tinypng.com/).
