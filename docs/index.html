One app for AI Markdown.

Live file watching

Open a file once and keep reading as an AI tool streams changes into it.

Smart edit mode

Edit the rendered document without staring at Markdown syntax, then save clean Markdown back to disk.

Rich Markdown support

Tables, tasks, alerts, Mermaid diagrams, math, code highlighting, tabs, and outlines are built in.

Small desktop stack.

md-reader uses Tauri 2, Rust, Svelte 5, comrak, CodeMirror, KaTeX, and Mermaid. It stays much smaller than an Electron app while feeling native enough for daily reading.

Local files

No account, no cloud workspace, and no telemetry.

Cross-platform

Release builds target Windows, macOS, and Linux.

Open-source

The code and releases are public on GitHub under MIT.
